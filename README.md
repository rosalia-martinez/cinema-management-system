ENGLISH VERSION
    Cinema Management System (Java)

    A console application for cinema seat management using 2D arrays, developed as part of my software development studies.

    Features

        Interactive seat map display
        Seat booking system
        Room occupancy tracking
        Simple console-based interface

    Quick Start

        Prerequisites
            Java JDK 8 or higher

        Installation & Running
        bash

        # Clone this repository
        git clone https://github.com/rosalia-martinez/cinema-management-system.git

        # Navigate to project folder
        cd cinema-management-system

        # Compile
        javac Gestion_Sala_De_Cine.java

        # Run
        java Gestion_Sala_De_Cine

    Project Structure

        cinema-management-system/
        ├── Gestion_Sala_De_Cine.java           # Main program with all functionality
        ├── README.md                           # This documentation
        └── .gitignore                          # Git configuration for Java projects

    Concepts Demonstrated

        2D array manipulation for seat mapping
        Console input/output handling
        Basic program flow control
        Object-oriented principles
